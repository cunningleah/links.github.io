nc-cms is a website Content Management System (CMS) designed with the primary goals of being user friendly and quick to integrate, while retaining the most important features of a modern day CMS. nc-cms is an open source and free to use project licensed under the zlib/libpng license. nc-cms was written in PHP and will run on any web server that supports PHP 5 or higher.
